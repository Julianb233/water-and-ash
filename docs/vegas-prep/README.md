# Vegas Convention Prep — Water & Ash Burials

Materials for the Cremators Convention in Las Vegas.

## Contents

| File | Description |
|------|-------------|
| `DEMO-SCRIPT-3MIN.md` | 3-minute elevator pitch for booth conversations |
| `DEMO-SCRIPT-10MIN.md` | 10-minute full walkthrough for interested prospects |
| `PARTNERSHIP-AGREEMENT.md` | Referral partnership agreement template (draft) |
| `CO-MARKETING-FLYER.html` | Print-ready co-marketing flyer (open in browser, print to PDF) |
| `OFFLINE-BACKUP.md` | Offline backup strategy and screenshot capture guide |

## Quick Start

1. **Review demo scripts** — practice both the 3-min and 10-min versions
2. **Print the flyer** — open `CO-MARKETING-FLYER.html` in Chrome, print to PDF or directly to printer
3. **Print partnership agreements** — 15 copies recommended
4. **Capture offline screenshots** — follow `OFFLINE-BACKUP.md` instructions
5. **Practice objection handling** — reference sheet is in the 3-min script

## Printing the Flyer

```bash
# Open in browser
open docs/vegas-prep/CO-MARKETING-FLYER.html

# Print settings: Letter size, no margins, background graphics ON
```
